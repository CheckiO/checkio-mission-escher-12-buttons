You should analyze the input data as the 2-dimensional array. Each element should be checked to the equity of 0. If the element == 0, you should check the next element. If not - you start checking the nearest numbers: from top, bottom, left and right of the current number. In that way you will find all numbers which form a geometry figure.
What should I do next with this numbers?
Each set of numbers which form an individual geometry figure you have to allocate to the individual structure. For example, you could create a list of lists and each internal list will be the list of all numbers from the one geometry figure.
Done. What's next?
At the end you need to count the sum of numbers in each list and then sort them in the descending order.
