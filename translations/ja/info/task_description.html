あなたは回廊を奥へ進むほどに幅が狭く天井が低くなっていくのに驚きましたがそれもすぐに終わりました。 なぜならむき出しの壁で行き止まりになったからです。 その壁には先に進むための手掛かりは何もありませんでした。 あなたは天井にいくつかの数字が描かれているのを見付けました。 これを操作することで先に進めるかもしれません。

いくつかの数字が書かれた幾何学的な形状のボタンが天井にいくつかあります。 あなたの仕事は正しい順番でそれらを押すことです。 価値がもっとも大きいものから始めて一番小さいものが最後です。 ボタンの価値はそれに描かれた数字の合計です。 一つのボタンは互いに上下左右に接している数字で構成されています。 あなたの関数はボタンを押す順番をボタンの価値のリストとして返さなくてはなりません。 0 は各ボタンを区分します。 もしいくつかのボタンが同じ価値を持つ場合はその順は任意です。

入力: 天井の数字を表す複数行の文字列

出力: ボタンの正しい順番のリスト
