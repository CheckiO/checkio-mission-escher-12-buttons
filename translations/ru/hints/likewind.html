Вам необходимо рассматривать исходные данные комплексно, как двумерный массив. Каждый элемент сперва следует проверить на равенство 0. Если элемент равен 0 - вы переходите к следующему. Если не равен - начинаете проверять его ближайших соседей: сверху, снизу, слева и справа. Таким образом вы определите цифры, принадлежащие к одной геометрической фигуре.
Что мне затем делать с полученными значениями?
Каждый набор цифр, входящих в отдельную геометрическую фигуру, вам необходимо выделить в отдельную структуру. Например, вы можете создать список списков, где каждый из внутренних списков, будет содержать набор цифр, принадлежащих одной фигуре.
Готово. Что теперь?
Теперь вам осталось лишь посчитать сумму цифр для каждого из списков и отсортировать полученные суммы по убыванию.
